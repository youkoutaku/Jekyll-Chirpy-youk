## Initial setting

```
xhost +
```
>Setting to enable to  use vscode. rviz ... in docker

## Create a Container
```bash
sudo docker run -it -v <local path>:/data \
	--device=/dev/dri \
	--group-add video \
	--volume=/tmp/.X11-unix:/tmp/.X11-unix \
	--env="DISPLAY=$DISPLAY" \
	--env="QT_X11_NO_MITSHM=1" \
	--name=<Container name> \
	<Image name>:<Tag>
	/bin/bash
```

### Check host-connect filer
```
cd /data
```

```

```


## Run a Container
### Start Container
```
sudo docker start -ia <Container ID>
```

### Check Running Container
```
docker ps
```

### Check all container
```
docker ps -a
```

### Check images
```
docker images
```
