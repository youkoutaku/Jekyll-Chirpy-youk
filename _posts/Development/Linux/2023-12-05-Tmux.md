## Run tmux
```

```

## Exit tmux
```
Ctrl-b + d
```

## List
```
tmux list-sessions
```

## Hotkey

### Horizontal Split window
```

```

### Vertical Split window
```
Ctrl-b + "
```

### Switch Window
